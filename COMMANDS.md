# Команды управления (обычный текст, без slash-команд)

Никаких `/dashboard` и `/digest-panel` — slash-команды кэшируются в рамках сессии
и не подхватывают правки файлов на лету. Работает только обычный текст,
проверено на практике.

| Что нужно | Что написать |
|---|---|
| Найти новости под критерии брифа | «Найди новости по Audi» / «Найди новости для дайджеста» |
| Показать найденных кандидатов | «Покажи кандидатов» |
| Переписать конкретный пост | «Перепиши пост p-2026-07-08-1» |
| Запланировать публикацию | «Запланируй пост p-2026-07-08-1 на 14:30» |
| Посмотреть статус/очередь | «Покажи дашборд» / «Покажи очередь на сегодня» |
| Опубликовать прямо сейчас | «Опубликуй сейчас» / «Запусти publisher.py» |
| Посмотреть историю публикаций | «Покажи историю за неделю» |
| Посмотреть логи | «Покажи последние логи» |

Я читаю `pipeline_state.json`, `used-news.md` и `publish_log.txt` заново
при каждом запросе — данные в ответах всегда актуальные на момент вопроса.
